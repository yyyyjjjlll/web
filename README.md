# 前端学习资料

本文件夹包含前端开发学习相关的资料和笔记。

## 📁 文件说明

| 文件名 | 描述 |
|--------|------|
| 前端.pdf | 前端常见知识点与八股文 |
| HTML基础知识思维导图.md | HTML 基础知识整理（思维导图格式） |
| 学习记录与错误修正8.docx | 学习过程中的记录与问题修正笔记 |
| lc记录.docx | LeetCode 刷题记录 |
| test.ts | TypeScript 算法题测试脚本 |
| git操作.txt | Git 常用操作命令记录 |

## 📌 使用说明

- **前端.pdf**：前端基础知识学习资料
- **HTML基础知识思维导图.md**：HTML 知识点结构化整理
- **学习记录与错误修正8.docx**：记录学习过程中遇到的问题和解决方案
- **lc记录.docx**：LeetCode 算法题解记录
- **test.ts**：运行 `npx tsx test.ts` 测试算法题
- **git操作.txt**：Git 命令速查
