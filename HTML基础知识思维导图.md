# 🌐 HTML 基础知识思维导图

## 📌 什么是 HTML？

```
HTML (HyperText Markup Language)
        │
        ├── 超文本标记语言
        │
        └── 用来告知浏览器如何组织页面的标记语言
```

### 核心知识点

| 序号 | 要点 |
|------|------|
| 1 | 标记也称为**标签**（元素） |
| 2 | 大小写都可以，建议**小写** |
| 3 | HTML 由一系列的**标签**组成 |

> 💡 **简单说：学习 HTML 就是学习标签**

---

## 📄 HTML 文档结构

```
<!DOCTYPE html>                    ← 文档类型声明
<html lang="zh-CN">               ← 根元素
    ┌─────────────────────────┐
    │ <head>                  │   ← 头部元素（元数据）
    │     <meta charset="UTF-8">
    │     <meta name="viewport" content="width=device-width, initial-scale=1.0">
    │     <title>页面标题</title>
    │ </head>                 │
    ├─────────────────────────┤
    │ <body>                  │   ← 页面内容
    │     ...显示内容...       │
    │ </body>                 │
    └─────────────────────────┘
</html>
```

### 各元素详解

| 元素 | 作用 | 说明 |
|------|------|------|
| `<!DOCTYPE html>` | 文档类型 | 告诉浏览器使用 HTML5 规范 |
| `<html>` | 根元素 | 包裹页面所有内容，`lang` 指定语言 |
| `<head>` | 头部元素 | 存放元数据，供机器处理 |
| `<meta charset>` | 字符集 | UTF-8 支持大多数语言字符 |
| `<meta viewport>` | 视口 | 移动端页面适配 |
| `<title>` | 标题 | 显示在浏览器标签页 |
| `<body>` | 主体 | 页面所有可见内容 |

### lang 属性

- `en` → 英语
- `zh-CN` → 简体中文

## 📝 标题和段落

### 标题标签 h1 ~ h6

```html
<h1>这是h1标题</h1>    ← 最大
<h2>这是h2标题</h2>
<h3>这是h3标题</h3>
<h4>这是h4标题</h4>
<h5>这是h5标题</h5>
<h6>这是h6标题</h6>    ← 最小
```

**特点：**
- ✅ 文字**加粗**显示
- ✅ 每行只显示**一个**（独占一行）
- ⚠️ `<h1>` 每个页面**只使用一次**（唯一性）
- 📌 每页标题层级**不超过三个**

### 段落标签 p

```html
<p>这是一个段落</p>
```

**特点：**
- 每行只显示一个
- 文字会自动换行
- 段落间有间距

## 💪 强调与重要性标签

HTML 提供了相应的标签，使其具有加粗、倾斜、下划线等效果，来着重**强调**某些文字。

### 语义化标签 ✅ 推荐使用

| 标签 | 作用 | 示例 |
|------|------|------|
| `<strong>` | **加粗** | `<strong>重要文字</strong>` |
| `<em>` | *倾斜* | `<em>强调文字</em>` |
| `<ins>` | <ins>下划线</ins> | `<ins>插入文字</ins>` |
| `<del>` | ~~删除线~~ | `<del>删除文字</del>` |

### 无语义标签 ❌ 不推荐

| 标签 | 作用 |
|------|------|
| `<b>` | 加粗 |
| `<i>` | 倾斜 |
| `<u>` | 下划线 |
| `<s>` | 删除线 |

> 💡 **尽量使用带有语义的标签！**

## 🧱 块级元素和内联元素

在 HTML 中有两种元素类别：**块级元素**和**内联元素**。分类不同决定着展示形式不一样。

```
              HTML 标签分类
                    │
        ┌───────────┴───────────┐
        │                       │
   ┌────▼────┐            ┌─────▼─────┐
   │ 块级元素 │            │  内联元素  │
   │ (Block) │            │ (Inline)  │
   └────┬────┘            └─────┬─────┘
        │                       │
   独占一行                 一行可放多个
```

### 块级元素 (Block)

| 特点 | 说明 |
|------|------|
| 独占一行 | 每个块级元素都从新行开始 |
| 可嵌套 | 可以嵌套其他元素 |
| 常见标签 | `<p>`, `<h1>`~`<h6>`, `<div>` 等 |

### 内联元素 (Inline)

| 特点 | 说明 |
|------|------|
| 一行多个 | 可以一行放多个，通常与文本一起使用 |
| 嵌套限制 | **不能**嵌套块级元素，可以嵌套其他内联元素 |
| 常见标签 | `<strong>`, `<em>`, `<a>` 等 |

### ⚠️ 注意事项

1. 段落 `<p>` 标签里面**不要**放其他块级元素
2. 段落里面主要放文字相关，比如内联元素
3. 元素分类不是一直不可更改的，后面可以通过 CSS 的 `display` 改变显示模式

## 🖼️ HTML 中的图片

### img 标签

```html
<img src="图片地址" alt="备选文本">
```

**特点：** 单标签（空元素）

### 图片属性总结

| 属性 | 作用 | 说明 |
|------|------|------|
| `src` | 图像地址 | 指向要插入到页面的图像地址 |
| `alt` | 备选文本 | 图片无法显示时提示的文字 |
| `width` | 宽度 | 设置图片宽度（建议 CSS 修改） |
| `height` | 高度 | 设置图片高度（建议 CSS 修改） |
| `title` | 图像标题 | 鼠标悬停显示的文本 |

**示例：**
```html
<img src="pig.png" alt="猪爸爸" title="猪爸爸出丑啦">
```

### ⚠️ 属性注意事项

1. 属性采取**键值对**形式：`属性="值"`
2. 属性之间没有必要的先后顺序
3. 属性之间必须有**空格分隔**

## 📸 常见图片格式

| 格式 | 特点 | 适用场景 |
|------|------|----------|
| **JPEG/JPG** | 有损压缩，放大缩小会变得模糊或有锯齿 | 摄影照片、网页图片（非透明背景） |
| **PNG** | 无损压缩，支持透明度 | Logo、网页图形、需要透明度的图像 |
| **GIF** | 支持动画，最多 256 色（索引色）| 简单动画、表情包、低色彩图形 |
| **WebP** | Google 开发，支持有损/无损压缩、透明度和动画 | 网页优化（替代 JPEG/PNG/GIF） |
| **AVIF** | 基于 AV1 视频编码，支持高压缩率和 HDR，压缩效率优于 WebP | 未来网页优化，需高性能压缩（B站、京东等） |

### 📋 图片格式选择指南

```
选择图片格式
     │
     ├── 🌐 网页优化 → 优先 WebP/AVIF（兼容时），备选 JPEG/PNG
     │
     ├── 🔲 透明图像 → PNG（静态）或 WebP（动态）
     │
     ├── 🎬 动画 → WebP/GIF（简单动画）
     │
     └── 📦 其他格式 → SVG、HEIC 等
```
